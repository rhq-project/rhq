What's new?

Versioned Deployments

AS7/EAP6/WildFly plugin now supports Versioned Deployments. A Deployment (EAR, WAR, etc) can be a managed server deployment or standalone deployment. This includes Versioned Subdeployments (WAR, EJB JAR inside an EAR, etc). A Versioned Deployment is one that incorporates a version into the artifact name. See more..

Enhanced Remote Agent Installation

The Administration>Agent GUI page now provides a way to install new agents on remote machines over SSH. It also now provides a way to start, stop and even uninstall an agent on a remote machine over SSH

Bundle support for AS7/EAP6 domain mode

The bundle content handover feature has been added and it purpose is to let the bundle target resource component participate in the bundle deployment. This new feature opened the door to bundle support of AS7 / EAP6 domain mode.
